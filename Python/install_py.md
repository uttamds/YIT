# Steps to Install Python and Get Started on Windows

## Step 1: Download Python

1. Open any browser
2. Go to **[https://www.python.org](https://www.python.org)**
3. Click **Downloads**
4. Download the **latest Python for Windows**

---

## Step 2: Install Python

1. Double-click the downloaded **`.exe`** file
2. **✔ Tick “Add Python to PATH”** (VERY IMPORTANT)
3. Click **Install Now**
4. Wait for installation to complete
5. Click **Close**

---

## Step 3: Verify Installation

1. Open **Command Prompt**
2. Type:

   ```
   python --version
   ```
3. Python version should be displayed

---

## Step 4: Create Your First Python Program

1. Open **Notepad**
2. Type:

   ```python
   print("Hello World")
   ```
3. Click **File → Save As**
4. Save as:

   ```
   hello.py
   ```

   (Select **All Files**)

---

## Step 5: Run the Python Program

1. Open **Command Prompt**
2. Navigate to the file location using `cd`
3. Run:

   ```
   python hello.py
   ```

---

## Step 6: Install a Code Editor (Recommended)

Install any one:

* **VS Code**
* **PyCharm**
* **IDLE (comes with Python)**

Use the editor to:

* Create `.py` files
* Write Python code
* Run programs easily

---

## Step 7: Start Learning Python Basics

Begin with:

1. Print & Input
2. Variables & Data Types
3. Conditions (`if-else`)
4. Loops
5. Functions

---

## Quick Summary

1. Download Python
2. Install (Add to PATH)
3. Verify installation
4. Create `.py` file
5. Run using Command Prompt
